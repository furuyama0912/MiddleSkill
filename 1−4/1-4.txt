①Javaの開発で必要なものは何ですか
 JDKとEclipse

 JDKと統合開発環境（IDE）のインストールが必要になります。


③ Javaを開発するための統合開発環境（IDE）は何ですか？
 Eclipse

 32ビットと64ビットのものがある


⑤環境変数とは何ですか？
OSが設定値などを永続的に保存し、利用者や実行されるプログラムから設定・参照できるようにしたもの。プログラムの実行時などに必要となる、利用者やコンピュータごとに内容が異なる設定値などを記録するために用いられる。
 
 
⑥JDKに含まれるものは何がありますか？（4つ以上）
 JRE（Java Runtime Environment）・・・Java実行環境
コンパイラ・・・プログラムを実行形式に変換する機能
デバッカ・・・バグの発見や修正を支援する機能
クラスライブラリ・・・開発を効率化するライブラリ
ドキュメント生成・・・ソースからドキュメントを自動的に作る機能


⑦JREとは何ですか？
 Java実行環境

 .classファイル

⑫Jarファイルとは何ですか？
Java プログラムの実行に必要なクラスファイルや設定ファイルがまとめられているアーカイブ。 Javaで作成したソフトウェアを配布する際に関連するファイルを1つにパッケージングする際に利用され、実際に多くのクラスライブラリはこの形式で配布されている。



⑬Warファイルとは何ですか？
 Java 製のWebアプリで利用されるクラスファイル(servlet)、設定ファイル、JSPやHTMLファイル、JAR形式のライブラリなどがまとめられているアーカイブ。


⑭デプロイとは何ですか？
システムにおいて「実行ファイルを実際のWebサーバー上に配置して、利用できる状態にすること」を指します。



⑮ビルドとは何ですか？
「プログラミングファイルを実行ファイルへ置き換える作業」のことです。



⑯デプロイはどのようなタイミングで行いますか？
企業によって異なり「アプリケーションのリリースタイミングの直前」
「システムの各機能検証が終わって、全体を検証できるようになったタイミング」で行われます。



⑰デプロイを実施する上で大切なことを3つ上げなさい（記事参照）
・なるべき早く終わらせること
・確実に行えるようにすること
・安定した状況下で行うこと
